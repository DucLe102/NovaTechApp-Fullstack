# NovaTechApp - Fullstack E-commerce Application

## Introduction
NovaTechApp is a fullstack shopping application that includes a web admin dashboard and a mobile client.

## Tech Stack
- Backend: Node.js, Express.js
- Database: MongoDB
- Frontend: ReactJS
- Mobile: React Native

## Features
- User authentication (Login/Register)
- Product management
- Category management
- Mobile shopping interface
- Profile management

## Project Structure
backend/   -> REST API server  
frontend/  -> Admin dashboard (React)  
mobile/    -> Mobile app (React Native)
